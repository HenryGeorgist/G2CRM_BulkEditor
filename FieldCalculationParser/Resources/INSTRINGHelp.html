INSTRING Function

The INSTRING function returns the integer value of the starting position of a string of text contained in another string of text. The first character in the string of text is position zero. If the string is not found in the string being searched the function results in a -1.

Syntax

INSTRING(stringToSearch,stringToFind)

  • stringToSearch This must be an expression that produces some sort of string (text) result. If the value that you want to search is numerical, convert it to a string using the ConvertToString() function.
  • stringToFind This must be an expression that produces some sort of string result. If the value that you want to search for is numerical, convert it to a string using the ConvertToString() function.

Example


 Formula
Result
1
 INSTRING(abcdef,def)
 3 (The first character of the string being searched starts at position zero).
2
 INSTRING('0123456','5')
 5 (The first character of the string being searched starts at position zero).
3
 INSTRING('Water','Rock')
 -1
4
 INSTRING('Water','wat')
 -1 (Capitalization matters)

Example 2

Select every structure that has a Found_Type of "Slab".



The easies way to do this is to simply say [Found_Type]="Slab". This will select every structure with a foundation type of "Slab".



What if, however, you want to select every "RES1" structure? Notice that we cannot use the method above because each "RES1" structure has a different name.



This is where the INSTRING function can be very useful. This function, however, returns an integer value. It will be necessary to use an IF statement to convert the integer into a boolean value.



Use the IF function to return a value of "True" if the INSTRING function returns any number greater than -1, and a value of "False" otherwise.

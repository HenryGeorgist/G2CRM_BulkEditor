= Function

The equals logic operator compares two expressions. If the expressions are equal, it will return true.

Syntax

expression1=expression2

  • expression1 and expression2 can be any expression: text expressions, logical expression, or numeric expression. If the output type for expression1 and expression2 differ, the result will always be False.

Example

Sample Data:
A = 4
B = 2

Formula
Result
1
 A=B False (4 does not equal 2)
2
 A="A" False (in this situation A is the equivalent of 4 while "A" is a string literal A.)

!= Function

A Not Equal operator (!=) is a logic operator which compares two expressions. If their result values are the same, it will return false. If their result values are different, it will return true.

Syntax

expression2 != expression1

The expression1 can be any expression, text expressions, logical expressions, or numeric expressions. If the output type for expression1 and expression2 differ, the result will always be True.
The expression2 can also be any expression, text expressions, logical expressions, or numeric expressions.

Example

2 != 2 = False

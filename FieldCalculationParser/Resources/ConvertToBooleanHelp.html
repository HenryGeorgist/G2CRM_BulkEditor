ConvertToBoolean Function

The BOOL function returns a true or false value.

Syntax

Bool(value)
  • value This can be anything that can be evaluated to either true or false.

Example


 Formula
Result
1
 BOOL(Val_Struct>100000)
 True if the value of the structure is greater than 100,000.
2
 BOOL(OccType=SFR1) True if the occupancy type is "SFR1".
4
 BOOL(2*2=4) True
3
 BOOL(True) True
4
 BOOL(False) False

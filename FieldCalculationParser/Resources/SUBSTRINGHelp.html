SUBSTRING Function

The SUBSTRING function returns a string of characters contained in another string.

Syntax

SUBSTRING(text,start_pos,num_characters)

  • text This must be an expression that produces a string result.

  • start_pos is the numeric value where you want to start extracting characters. The first character in the string is position 0 (zero).

  • num_characters is the number of characters that will be extracted.

Example


 Formula
Result
1
 SUBSTRING(abcdef, 2, 3)
 cde
2
 SUBSTRING(abcdef, 0, 2)
 ab

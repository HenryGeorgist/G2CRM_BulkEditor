ConvertToDouble

A DBL returns a larger float type that holds both bigger and more precise numbers. This data type should be used when a high degree of accuracy is desired.

Syntax

Dbl(value)

  • value This must be an expression that produces some sort of numerical result.

Example


 Formula
Result
1
 Dbl(25) 25
2
 Dbl(2.31515467891234) 2.31515467891234
3
 Dbl(12345678.12345678) 12345678.12345678
4
 Dbl(2.73*3.218) 8.78514

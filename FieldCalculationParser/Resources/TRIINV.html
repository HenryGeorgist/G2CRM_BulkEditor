TRIINV Funtion

Returns a random integer number between the numbers you specify. A new random integer number is returned every time the worksheet is calculated

Syntax

TRIINV(random_number,min,most_likely,max)

  • random_number is a decimal value between 0 and 1. Rand() is commonly used here.
  • min is the minimum integer Triinv could return.
  • most_likely is the expected, or most likely, value
  • max is the largest integer Triinv could return.

Example


 Formula Description/Result
1
 TRIINV(RAND(),1,2,3) Random number between 1 and 3 and most likely value of 2 (varies)
2
 TRIINV(RAND(123),1,2,3) 2.82425538546691 This result is a constant because the RAND() function has been given a seed. If left blank, the RAND() function uses the computer's clock as the seed resulting in a differnt number every time.
3
 TRIINV(.5,0,500,1000) 500

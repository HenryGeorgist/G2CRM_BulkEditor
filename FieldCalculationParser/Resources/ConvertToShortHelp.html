ConvertToShort

The ConvertToShort function converts numerical data into the short data type. Shorts are stored using only 2 bytes. This creates less memory consumption, but also limits the range of values that can be stored as a short to -32,768 through 32,767. When converting numbers to the short data type make sure that no values exceed this range or an exception will be thrown. Numbers with decimal values will have the decimal portion truncated. This means that 2.99 will be converted to 2.

Syntax

Short(value)

  • value  This must be an expression that produces some sort of numerical result.

Example


 Formula
Result
1
 Short(87) 87
2
 Short(32.89) 32
3
 Short(5.5) 5
4
 Short(33000) 0 (numbers that are outside the range receive a value of zero)

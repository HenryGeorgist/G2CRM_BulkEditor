>= Function

A >= logic operator compares two expressions. If the first expression is larger than or equal to the second expression, it will return true.

Syntax

expression1>= expression2

The expression1 must be any logical expression, which means the expression must yeild either a result of "True" or "False".
The expression2 must aslo be any logical expression.

Exampel

12>= 1 It will return a Boolean value “True"

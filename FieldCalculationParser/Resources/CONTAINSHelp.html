CONTAINS Function

The CONTAINS logic operator compares two character strings. If the first character string contains the second character sequence, it will return "True". A great time to use the CONTAINS operator is when selecting rows that contain a specific text value.

Syntax

Contains(expression1,expression2)

  • expression1 must be any logical expression, which means the expression must yeild either a result of "True" or "False".
  • expression2 must aslo be any logical expression.

Example


 Formula
Result
1
 Contains(abcdefg,def) True
2
 Contains(ENGLAND,england) True
3
 Contains(abcd,xyz) False

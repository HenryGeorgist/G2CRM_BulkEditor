ROUNDDOWN Function

Rounds a number down to the nearest integer.

Syntax

ROUNDDOWN(Number)
  • Number is any real number that you want rounded down.

Example


 Formula
 Result
1
 ROUNDDOWN(3.2)
 3
2
 ROUNDDOWN(76.9) 76
3
 ROUNDDOWN(3.14159) 3
4
 ROUNDDOWN(-3.14159) -4
5
 ROUNDDOWN(31415.92654) 31415

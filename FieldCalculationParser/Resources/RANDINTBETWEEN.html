RANDINTBETWEEN Funtion

Returns a random integer number between the numbers you specify. The minimum value is inclusive, while the maximum number is exclusive. A new random integer number is returned every time the worksheet is calculated

Syntax

RANDINTBETWEEN(min,max,[seed])

  • min is the smallest integer RANDINTBETWEEN will return.
  • max is one larger then the largest integer RANDINTBETWEEN will return.
  • Seed is the seed for the random number generator (optional).

Example


 Formula Result
1
 RANDINTBETWEEN(1,100) Random number between 1 and 99
2
 RANDINTBETWEEN(-1,1) Either -1 or 0

ConvertToString Function

The ConvertToString function converts data to the string data type. Although quotation marks are not required, it is a good idea to use them if the string literal is desired (see examples 4 and 5 below).

Syntax

Str(value)
  • value This can be any expression.

Example


 Formula
Result
1
 String("65") 65
2
 String(65) 65
3
 String(Hello) Hello
4
 String(2+2) 4
5
 String("2+2") 2+2

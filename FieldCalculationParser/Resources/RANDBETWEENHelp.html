RANDBETWEEN Funtion

Returns a random number between the numbers you specify. A new random number is returned every time the worksheet is calculated

Syntax

RANDBETWEEN(min,max,[seed])

  • min is the smallest number RANDBETWEEN will return.

  • max is the largest number RANDBETWEEN will return.

  • seed is an optional seed value for the random number generator.

Example


 Formula Result
1
 RANDBETWEEN(1,100) Random number between 1 and 100
2
 RANDBETWEEN(-1,1) Random number between -1 and 1
3
 RANDBETWEEN(1,10,5) 4.04532856822262

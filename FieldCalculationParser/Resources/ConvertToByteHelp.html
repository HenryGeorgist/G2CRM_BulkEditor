ConvertToByte

Convert to byte stores numeric values with 8 bits. This is the optimal storage size for integers ranging from 0 to 255. Numbers with decimal values are converted to zero. If there is a chance that the data will be greater than 255 or less than 0 or have decimal values use a different data type.

Syntax

Byte(value)

  • value This must be an expression that produces a numerical result ranging from 0 to 255.

Example


 Formula
Result
1
 Byte(109) 109
2
 Byte(5) 5
3
 Byte(300) 0 The result is zero (0) if the value is outside the range (0-255).

RANDINTBETWEEN Funtion

Returns a random integer number between the numbers you specify. The random number could be the first number you specify up to the last number minus one. A new random integer number is returned every time the worksheet is calculated.

Syntax

RANDINTBETWEEN(min,max,[seed])

  • min is the smallest integer RANDINTBETWEEN will return.

  • max is one higher than the largest integer RANDINTBETWEEN will return.

  • seed is an optional seed value for the random number generator.

Example


 Formula Result
1
 RANDINTBETWEEN(1,100) Random number between 1 and 99
2
 RANDINTBETWEEN(-1,1) Random number between -1 and 0
3
 RANDINTBETWEEN(1,3) Either 1 or 2
4
 RANDINTBETWEEN(1,10,5) 4

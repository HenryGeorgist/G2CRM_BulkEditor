ConvertToInteger

The ConvertToInteger function converts numerical values to the integer data type. The integer data type stores integer numbers using 4 bytes of memory. This amount of memory leads to optimized performance, but has a narrower range of values than other data types. Integer values can range from -2,147,483,648 through 2,147,483,647. When converting numbers to the integer data type make sure that no values exceed this range or an exception will be thrown. Numbers with decimal values will have the decimal portion truncated. This means that if the number 2.99 is being converted to an integer, the resulting value will be 2.

Syntax

Int(value)

  • value This must be an expression that produces some sort of numerical result.

Example


 Formula
Result
1
 Int(65) 65
2
 Int(32.89) 32
3
 Int(5.2) 5
